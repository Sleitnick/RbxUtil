Spring

Simulates a critically damped spring. This is mostly just a wrapper around TweenService:SmoothDamp().

Functions​

new​

</>
Spring.new(
initial: T,--

Initial value

smoothTime: number,--

Approximate time it takes to reach target

maxSpeed: number--

Maximum speed (defaults to infinity)

) → Spring<T>

Creates a new spring.

The type T can be a number, Vector2, Vector3, or CFrame.

Update​

</>
Spring:Update(deltaTime: number) → T

Updates the spring. This method should be called continuously, even if the target value hasn't changed.

local newValue = spring:Update(dt)

</>

Impulses the spring.
